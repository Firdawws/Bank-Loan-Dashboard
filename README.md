# **Bank Loan Analysis Dashboard**

## Overview
This Power BI dashboard provides comprehensive analysis of bank loan data, enabling data-driven decision making for loan portfolio management.

## Dashboard Preview
<img width="787" height="450" alt="image" src="https://github.com/user-attachments/assets/d6252771-70f1-41c5-bece-1e33374bcbc4" />


## Key Features
- Loan approval rates and trends
- Portfolio performance metrics
- Customer demographics analysis
- Default rate monitoring
- Loan amount distribution by category

## Tools Used
- **Power BI Desktop** - Data visualization and dashboard creation
- **DAX** - Custom measures and calculations



## Insights
This dashboard helps identify:
- High-risk loan segments
- Seasonal lending patterns
- Customer creditworthiness trends
- Revenue opportunities
